# Stitch to Remotion Videos

## When to use this skill
- When requested to "create a walkthrough video" or "render app demo".
- To showcase Stitch screens with animated transitions and text overlays.

## Workflow

### 1. Data Gathering
- [ ] **Fetch Project**: Retrieve project and screen information via Stitch MCP.
- [ ] **Gather Assets**: Download screenshots for each scene.

### 2. Video Composition
- [ ] **Architecture**: Generate Remotion components for the video structure.
- [ ] **Transitions**: Apply smooth zooms, pans, and fades between screens.
- [ ] **Overlays**: Add text overlays based on screen descriptions.

### 3. Rendering
- [ ] **Preview**: Use Remotion's dev tools to verify the composition.
- [ ] **Execute Render**: Call the Remotion render command to produce the video file.

## Instructions
- **Visual Storytelling**: Focus on the "User Flow" rather than just showing static screens.
- **Timing**: Use 2-3 seconds per screen with 500ms transitions for optimal pacing.

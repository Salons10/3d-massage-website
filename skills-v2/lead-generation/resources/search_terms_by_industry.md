# Search Terms by Industry

Ready-to-use search term arrays for common local service industries. Use **3 terms**
per industry for best budget efficiency.

---

```json

```

Extended (5 terms, use with higher budgets):
```json
["roofing contractor", "roofer", "roof repair", "roof replacement", "metal roofing"]
```

## Usage Tips

1. **Start with 3 terms** — matches the budget formula in SKILL.md
2. **Extended terms** are useful when budget allows 200+ places
3. **Combine with location modifiers** if results are too broad (e.g., "pool cleaner near me" doesn't help — use the `locationQuery` parameter instead)
4. **Avoid generic terms** like "contractor" or "service" on their own — too many false positives
5. **Custom industries** — create your own by identifying the Google Maps category names used in your target market
